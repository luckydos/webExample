VSCode(Visual Studio Code) 단축키

CTRL + D

특정 문자열 선택(변수) 후 CTRL + D 를 누르면 동일한 문자열을 순서대로 선택하고, 커서를 이동 되고선택된 문자열을 한번에 수정 가능.
ESC 키를 누르면 선택 구역 해제

ALT + CLICK

ALT 키를 누른 상태에서 클릭시 클릭한 모든 구역이 선택되고, 한번에 문자열 수정 가능. ESC 키를 누르면 선택 구역 해제

ALT + (UP or DOWN)Arrow

영역 선택 후 이 키를 누르면 화살표 방향으로 이동. 여러줄 이동도 가능.

ALT + SHIFT + (UP or DOWN)Arrow

영역 선택 후 이 키를 누르면 화살표 방향으로 복사. 여러줄 복사도 가능.

CTRL + /

선택영역 주석 처리 (Line)

ALT + SHIFT + A

선택영역 주석 처리 (Block)

ALT + SHIFT + I

영역 선택 후 이 키를 누르면 각 라인의 끝선 기준으로 커서 입력. 멀티라인 편집 가능

ALT + SHIFT + Drag

ALT + SHIFT 누른 후 마우스 드래그 시 드래그하는 영역을 일괄로 선택.
